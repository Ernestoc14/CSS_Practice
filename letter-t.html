Introduccion:

En esta pagina se encuentran las palabras mas comunes con la letra S

  1. Tacos:

    Preparación culinaria muy popular de México que consiste en una tortilla, generalmente de maíz, que comúnmente se enrolla para contener dentro los ingredientes.

  2. Texto:

    Conjunto de enunciados que permite dar un mensaje coherente y ordenado, ya sea de manera escrita o a través de la palabra.

  3. Tiza:

    es una arcilla blanca que, preparada en barritas, se usa para escribir en las pizarras y, pulverizada, para limpiar metales y para hacer pinturas de fácil lavado. Para las elecciones individuales se usaba un pizarrín.

  4. Tronco:

    Principal elemento estructural de un árbol que soporta las ramas y todo el resto del mismo (hojas, flores, frutos, etc.). A su vez, las raíces se insertan en el suelo y le sirven de fundamento a todo el árbol.

  5. Taza:

    recipiente con una sola asa, usado básicamente para tomar líquidos calientes. En algunos países de Latinoamérica se denomina jarro o pocillo. Puede tener diversas formas, colores y tamaños, presentarse decorada o serigrafiada.
